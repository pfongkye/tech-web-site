HTML

HTML stands for HyperText Markup Language.

HTML is the language used to structure web pages.

CSS

CSS stands for Cascading Style Sheets.

CSS is used to apply visual representations to HTML elements.

JavaScript

JavaScript is a programming language.

JavaScript is used to add dynamic behaviors to web sites.

React

React is a JavaScript library.

React is used to build user interface components.

Data Structures

Data structures are used to store data in a given way.

Data structures go hand in hand with algorithms.
