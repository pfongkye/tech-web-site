HTML

HTML stands for HyperText Markup Language.

HTML is the language used to structure web pages.

CSS

CSS stands for Cascading Style Sheets.

CSS is used to apply visual representations to HTML elements. An example of ruleset: :root { box-sizing: border-box; }

JavaScript

JavaScript is a programming language.

JavaScript is used to add dynamic behaviors to web sites. A declaration of a variable can be as follows var i;. And for a function: function myFunction(){console.log("Hello world.");}

React

React is a JavaScript library.

React is used to build user interface components. An example of a component is function MyComponent(){return React.createElement("div",null,"Hello world.");}
Some features/keywords:

  • Hooks
  • One way binding

Data Structures

Data structures are used to store data in a given way. An example is var array = [];

Data structures go hand in hand with algorithms.
